# porcupine

![](docs/logo.png)

Go bindings for wake word detection library.

## Features

- Porcupine 1.9.
- Windows, Linux and Mac.
- Embeds the model and dynamic libraries into the executable.
- Custom trained wake words work indefinitely.

## Requirements

Requires Go 1.16 and above.
